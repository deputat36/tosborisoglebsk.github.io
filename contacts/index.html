Контакты и как прислать исправления

Портал задуман так, чтобы данные было легко обновлять. Ниже — понятная инструкция.

Контакты портала

  • Эта страница — шаблон. Вставьте сюда реальные контакты (почта/телефон/соцсети).
  • Можно добавить форму Google Forms (ссылкой) для приёма исправлений.

Как прислать исправление (без доступа к GitHub)

  1. Напишите: название ТОС, что именно исправить, и правильный вариант.
  2. Если меняются границы — приложите список улиц/домов (или ссылку на карту).
  3. Если меняются контакты — приложите актуальный телефон/почту/ссылки.

Как обновлять данные через GitHub (если доступ есть)

  • Каталог ТОС: data/tos.data.js
  • Новости: data/news.data.js
  • Документы: data/docs.data.js
  • Карта: data/tos.geo.js
Если хотите — добавлю “шаблон заявки на исправление” (Issue Template) в репозиторий, чтобы люди присылали данные строго по форме.

Инструмент импорта из Excel

Откройте: tools/import.html

  • Выберите Excel/CSV.
  • Получите готовый текст для data/tos.data.js.
  • Вставьте в файл и закоммитьте.
