# Публикация на GitHub Pages

1. Репозиторий должен быть публичным, ветка — `main`.
2. В настройках GitHub Pages выберите Source: **Deploy from a branch**, ветку `main`, папку `/ (root)`.
3. После каждого коммита убедитесь, что статика собирается без инструментов: сайт использует только HTML, CSS и JS из репозитория.
4. Для ручной проверки откройте `https://<owner>.github.io/<repo>/` или настроенный домен из файла `CNAME`.
5. Если обновляете домен — не забудьте синхронизировать `CNAME` и `sitemap.xml`.
