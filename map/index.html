Карта ТОСов

Минимальная версия: точки/границы из GeoJSON. Если данных нет — вы увидите инструкцию, как добавить.

Как добавить границы/точки

  1. Откройте файл data/tos.geo.js.
  2. Вставьте GeoJSON FeatureCollection в поле window.DATA.tosGeo.
  3. В properties укажите name, place, id (slug из каталога).

Если нужно — добавлю “инструкцию по рисованию границ” (через geojson.io) отдельной страницей.

Карта использует Leaflet через CDN (подгружается только если есть геоданные).
