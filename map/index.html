Карта ТОСов

Демо-версия с точками из каталога. Геоданные можно уточнить и расширить.

Что сейчас на карте

На карте показаны демо-точки ТОС, чтобы посетители могли увидеть, как будет выглядеть полноценно заполненный раздел.

Подробные инструкции по подготовке GeoJSON вынесены в файл docs/data-format.md репозитория.

Карта использует Leaflet через CDN (подгружается только если есть геоданные).
